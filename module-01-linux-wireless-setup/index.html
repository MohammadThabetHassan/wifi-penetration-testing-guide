02 OS Environment — VM vs. Bare Metal

Your choice of environment has a direct impact on injection reliability. Understand the trade-offs before committing.

💻 Virtual Machine

Pros
  • Easy snapshots / rollback
  • Runs on any host OS
  • Disposable environment
Cons
  • USB passthrough unreliable
  • Injection may fail/drop
  • Channel hopping broken
  • Internal adapter inaccessible

⚙ Bare Metal RECOMMENDED

Pros
  • Full hardware access
  • Reliable injection
  • Native kernel drivers
  • Best performance
Cons
  • Dedicated machine or dual-boot
  • Less convenient for daily use

💾 Live USB Boot

Pros
  • No installation needed
  • Full hardware access
  • Portable
Cons
  • No persistence (unless configured)
  • Slower than bare metal install
  • Wordlists lost on reboot

04 Core Wireless Commands

Identifying and Inspecting Interfaces

kali@kali:~$ — Interface discovery and inspection
# List all wireless interfaces kali@kali:~$ iw dev phy#0 Interface wlan0 type managed addr aa:bb:cc:dd:ee:ff channel 6 (2437 MHz) # Check if phy supports monitor mode kali@kali:~$ iw phy phy0 info | grep -A 10 "Supported interface modes" Supported interface modes: * IBSS * managed * AP * monitor ← Must be present! * P2P-client # Legacy tool — still useful for quick checks kali@kali:~$ iwconfig wlan0 wlan0 IEEE 802.11 ESSID:off/any Mode:Managed Frequency:2.412 GHz Access Point: Not-Associated Tx-Power=20 dBm Retry short limit:7

Fixing RF Kill & Blocked Adapters

kali@kali:~$ — rfkill troubleshooting
kali@kali:~$ rfkill list 0: phy0: Wireless LAN Soft blocked: no Hard blocked: no 1: phy1: Wireless LAN Soft blocked: yes ← blocked! fix with rfkill unblock Hard blocked: no # Unblock by index kali@kali:~$ sudo rfkill unblock 1 # Or unblock everything kali@kali:~$ sudo rfkill unblock all

Killing Interfering Processes

kali@kali:~$ — airmon-ng check kill
# Check what processes might interfere kali@kali:~$ sudo airmon-ng check Found 2 processes that could cause trouble. PID Name 743 NetworkManager 891 wpa_supplicant # Kill them all at once kali@kali:~$ sudo airmon-ng check kill Killing these processes... PID Name 743 NetworkManager 891 wpa_supplicant # Restore when done kali@kali:~$ sudo systemctl start NetworkManager

MAC Address Spoofing

kali@kali:~$ — MAC spoofing with macchanger
# Always bring interface down before changing MAC kali@kali:~$ sudo ip link set wlan0 down # Assign a random MAC address kali@kali:~$ sudo macchanger -r wlan0 Current MAC: 00:c0:ca:a1:b2:c3 (Alfa, Inc.) Permanent MAC: 00:c0:ca:a1:b2:c3 (Alfa, Inc.) New MAC: 36:4a:c9:8f:12:7e (unknown) # Bring interface back up kali@kali:~$ sudo ip link set wlan0 up # Restore original hardware MAC kali@kali:~$ sudo ip link set wlan0 down && sudo macchanger -p wlan0

06 Troubleshooting Common Issues

ERROR Interface not appearing after plugging in adapter
▶
Fix: adapter not detected
kali@kali:~$ dmesg | tail -20 # Look for USB attach messages and any driver errors kali@kali:~$ lsusb # If device shows in lsusb but no interface, try loading driver: kali@kali:~$ sudo modprobe 88XXau # For AR9271: kali@kali:~$ sudo modprobe ath9k_htc # RTL8812AU: plug into USB 3.0 port (blue port)
ERROR airmon-ng fails to enable monitor mode
▶
Fix: monitor mode failure
# Kill interfering processes first kali@kali:~$ sudo airmon-ng check kill # Try manual monitor mode as fallback kali@kali:~$ sudo ip link set wlan1 down kali@kali:~$ sudo iw dev wlan1 set type monitor kali@kali:~$ sudo ip link set wlan1 up kali@kali:~$ iwconfig wlan1 | grep Mode Mode:Monitor ← success
ERROR Packet injection test fails — "No answer from AP"
▶
Fix: injection test failure
# Lock to the AP's channel first kali@kali:~$ sudo iw dev wlan1mon set channel 6 # Then retry injection test kali@kali:~$ sudo aireplay-ng -9 wlan1mon Trying broadcast probe requests... Injection is working! # If still failing, try a powered USB hub or different port
ERROR Soft-blocked adapter — rfkill shows "blocked: yes"
▶
Fix: rfkill soft block
kali@kali:~$ rfkill list 1: phy1: Wireless LAN Soft blocked: yes kali@kali:~$ sudo rfkill unblock all kali@kali:~$ rfkill list 1: phy1: Wireless LAN Soft blocked: no Hard blocked: no
